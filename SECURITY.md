# Security Policy

## Supported Versions

Use the latest version of this project to ensure the best security and performance.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

This project is a static web application and does not store user data on any server. All data is encoded within the URL (client-side only).

However, if you discover a security vulnerability (e.g., XSS issue), please report it by:

1.  Opening an issue in the [Issues](https://github.com/Alperen012/happy_new_year/issues) tab.
2.  Describing the vulnerability and how to reproduce it.

Since this is a personal project, response times may vary.
